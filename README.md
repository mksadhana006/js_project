## Temperature Converter (Celsius ↔ Fahrenheit)

### Description

This project is a simple temperature converter that converts values between Celsius and Fahrenheit.
It is designed to demonstrate basic input handling and mathematical conversion logic.
No external libraries or backend services are used.


### Features

* Convert Celsius to Fahrenheit
* Convert Fahrenheit to Celsius
* User-friendly input fields
* Instant result display
* Simple and clean interface


### Technologies Used

* HTML5
* CSS3
* JavaScript


### Conversion Logic

* Celsius to Fahrenheit: `(C × 9/5) + 32`
* Fahrenheit to Celsius: `(F − 32) × 5/9`
